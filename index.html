1. 이차함수의 정의

함수 y = f(x)에서 y가 x에 관한 이차식

y = ax² + bx + c (a ≠ 0, a, b, c는 상수)

로 나타내어질 때, 이 함수를 이차함수라고 합니다.

주의: a가 0이면 이차함수가 아니며, 일차함수나 상수함수가 됩니다.

2. 이차함수 y = ax²의 그래프

그래프의 특징

  • 원점 (0, 0)을 꼭짓점으로 합니다.
  • y축을 축의 방정식(x=0)으로 합니다.
  • 모양은 포물선입니다.

상수 a의 역할

  • a > 0: 아래로 볼록
  • a < 0: 위로 볼록
  • |a|가 클수록: 그래프의 폭이 좁아짐

3. 이차함수 표준형

표준형: y = a(x - p)² + q

이차함수 y = ax²의 그래프를 x축 방향으로 p만큼, y축 방향으로 q만큼 평행이동한 것입니다.

  • 꼭짓점: (p, q)
  • 축의 방정식: x = p

4. 이차함수 y = ax² + bx + c

일반형을 표준형으로 고쳐서 그래프의 성질을 파악합니다.

y = ax² + bx + c
→ y = a(x - p)² + q
